How to run the Laravel CRUD  Project
1. Download the  zip file
2. Extract the file and copy laravel folder
3.Paste inside root directory(for xampp xampp/htdocs)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name laravel
6. Import laravel.sql file(given inside the zip package in for-import-database folder)
7.Run the script http://localhost/laravel/public/posts (frontend)
